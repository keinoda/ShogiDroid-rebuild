## v0.0.4

### クラウドエンジン
- クラウド画面を統合し、vast.ai に加えて GCP Spot VM / AWS Spot インスタンスの利用に対応
- リモート接続設定に SSH / TCP 切り替え、CPU コア数・RAM・GPU RAM 設定、GCP 稼働料金表示を追加
- インスタンス一覧取得の並列化、15 秒タイムアウト、IPv4 優先化などで接続まわりの安定性を改善
- クラウド Watchdog と自動起動まわりを整理し、リモートエンジン接続の復旧動作を改善

### UI / 操作
- 対局情報編集ダイアログをメイン画面から開けるように追加
- 盤面の左 1/3 タップで前の手、右 1/3 タップで次の手へ移動できるように対応
- ドロワーメニューをセクションごとに整理し、詳細メニューの表示切り替え設定を追加
- ステータスバー表示時のレイアウトと配色を調整

### 棋譜 / 解析
- 将棋ウォーズの棋譜一覧を取得して、そのままアプリに読み込める画面を追加
- コメント内の過去の解析結果表示、解析コメントの復元、一括削除に対応
- 推定選択率表示を改善し、PolicyMoveInfo に depth 1 の評価値を保持
- Hash 自動設定を端末コア数ベースに変更し、解析ページ構成を調整

### その他
- SSH 鍵インポート導線をアプリ設定へ移動し、公開鍵の自動生成を無効化
- ログ出力の欠落や空の `catch` ブロックを修正し、デバッグしやすさを改善
- 各種コードレビュー指摘の反映とコード整理を実施
