## v0.0.3

### アプリ基盤
- アプリID を `com.ngs43.shogidroid` に変更、名称を「ShogiDroid」に統一
- 設定のバックアップ・復元機能を追加（アプリID変更時の設定引き継ぎ対応）

### UI
- Material Design 風の UI に全面刷新（テーマ切替、角丸、カラー体系）
- 評価値の勝率表示を見やすく調整
- 解析読み筋ダイアログを Material 化し、棋譜への追加に対応
- 全画面表示時のステータスバー領域の色差を修正

### 解析機能
- Komoring 詰めろ解析と棋譜操作まわりを追加
- 並列解析をフォアグラウンド実行に変更（バックグラウンドサービスを廃止）
- バックグラウンド解析とエンジン状態管理を改善

### vast.ai クラウドエンジン
- interruptible（入札）モードに対応し、オファーカードに最低入札額を表示
- 解析終了後 5 分間操作がなければインスタンスを自動一時停止
- リモートエンジン接続失敗時にインスタンスを自動起動し解析を再開
- API レスポンス解析を修正（ステータスが unknown になる問題を解消）
- `intended_status` ベースの状態判定に変更（起動直後の exited 誤判定を修正）

### 棋譜読み込み
- 6shogi の棋譜読み込みに対応
- 棋神アナリティクス Deep Link 対応（将棋ウォーズ AI ボタンから直接読み込み）

### その他
- エンジンダウンロードページ（旧サイト）のリンクを無効化
